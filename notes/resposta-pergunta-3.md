# Pergunta 3 - Quais são os principais componentes do Databricks?

## Pergunta
Quais são os principais componentes da plataforma Databricks?

## Prompt utilizado
Com base na fonte [S1], quais são os principais componentes da plataforma Databricks?
Explique de forma objetiva.
Inclua a referência no formato [S1].

## Resposta do NotebookLM
Com base na fonte fornecida, os principais componentes e ferramentas que integram a plataforma Databricks incluem:
• Apache Spark: A base da plataforma, sendo um framework de computação distribuída de código aberto utilizado para processamento de dados.
• Delta Lake: Um projeto de código aberto que adiciona suporte a transações ACID aos data lakes, garantindo maior confiabilidade aos dados.
• Databricks SQL (anteriormente SQL Analytics): Ferramenta voltada para a execução de inteligência de negócios (BI) e relatórios analíticos diretamente sobre os data lakes, utilizando SQL padrão.
• MLflow e Koalas: Projetos de código aberto integrados para suporte às áreas de ciência de dados, engenharia de dados e aprendizado de máquina (machine learning).
• Conjunto de ferramentas Mosaic: Inclui o AI Vector Search (para modelos RAG), AI Model Serving (implantação e monitoramento de modelos) e AI Pretraining (para a criação de modelos de linguagem próprios).
• DBRX: Um modelo de linguagem de grande porte (LLM) de código aberto desenvolvido pela própria empresa.
• Agent Bricks: Uma suíte de ferramentas e plataforma de desenvolvimento para a criação de agentes de IA sem a necessidade de código (no-code).
• Lakebase: Um banco de dados transacional (OLTP) serverless integrado à plataforma.
• Databricks One: Uma plataforma de BI de inteligência artificial sem código.
• Redash: Uma ferramenta de código aberto para visualização de dados e construção de painéis interativos.
