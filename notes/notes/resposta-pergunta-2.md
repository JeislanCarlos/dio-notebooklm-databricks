# Pergunta 2 - O que é a arquitetura Lakehouse?

## Pergunta
O que é a arquitetura Lakehouse?

## Prompt utilizado
Com base na fonte [S1], explique o que é a arquitetura Lakehouse.
Responda apenas com base na fonte fornecida.
Inclua a referência no formato [S1].

## Resposta do NotebookLM
Com base nas fontes fornecidas, a arquitetura Lakehouse é um modelo de plataforma de dados em nuvem desenvolvido pela Databricks que combina elementos e funcionalidades de armazéns de dados (data warehouses) e lagos de dados (data lakes).
As principais características dessa arquitetura incluem:
• Gerenciamento Unificado de Dados: Permite a gestão tanto de dados estruturados quanto de dados não estruturados em um único ambiente.
• Flexibilidade de Esquema: Construída sobre o framework de código aberto Apache Spark, a arquitetura permite a execução de consultas analíticas em dados semiestruturados sem a exigência de um esquema de banco de dados tradicional
