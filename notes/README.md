# Anotações do projeto

Esta pasta reúne os registros produzidos durante o uso do NotebookLM como ferramenta de aprendizagem ativa no estudo da plataforma Databricks.

## Conteúdo das anotações

- **Respostas às perguntas (Perguntas 1 a 5):**  
  Cada arquivo registra uma pergunta estratégica formulada no NotebookLM, o prompt utilizado, a resposta gerada com base nas fontes fornecidas e a referência correspondente.

- **Resumo estruturado:**  
  O arquivo `resumo-estruturado.md` consolida os principais conceitos estudados, funcionando como um mini-guia de estudo sobre Databricks, Lakehouse, Apache Spark e Delta Lake.

## Metodologia

As perguntas foram elaboradas com o objetivo de explorar conceitos introdutórios, arquitetura, componentes, casos de uso e comparações conceituais.  
As respostas foram geradas no NotebookLM a partir de fontes abertas previamente selecionadas e documentadas na pasta `sources`.

Este conjunto de anotações representa o principal produto do projeto desenvolvido.
